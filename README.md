# Ecommerce AI Chatbot

An intelligent AI-powered chatbot designed for e-commerce platforms that enhances customer experience through **natural conversation, product understanding, and smart responses**.

This project integrates **LLM + RAG (Retrieval-Augmented Generation) + Vector Embeddings** to deliver accurate, context-aware responses to user queries.

---

## Features

- Natural language conversation (LLM-based)
- Context-aware answers using RAG
- Product-related query handling
- Vector embeddings for semantic search
- Fast API-based backend response system
- Clean and responsive chat UI
- Backend–Frontend integration

---

## How It Works

The system follows a modern AI pipeline:

### Step-by-step:

1. User sends a query  
2. Backend converts query → embeddings  
3. Similar context retrieved from vector database  
4. Context + query sent to LLM  
5. LLM generates final response  
6. Response displayed in chatbot UI
